How to Install

Install from the GitHub ZIP and load it as an unpacked extension.

  1. Download the latest source ZIP from GitHub: cleen main.zip
  2. Unzip the archive.
  3. Open chrome://extensions in Chrome.
  4. Enable Developer mode (top right).
  5. Click Load unpacked.
  6. Select the unzipped cleen-main folder.

Stack: Manifest V3, Chrome Processes API, local storage.

Privacy: All data stays on your device. No analytics, no tracking.
