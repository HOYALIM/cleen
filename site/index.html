Cleen automatically suspends inactive tabs to free up memory. Track usage, detect heavy sites, and keep Chrome running smooth.

Download ZIP How to Install

Auto-Suspend

Tabs inactive for 30+ minutes are automatically discarded. Wake them up with a single click.

Memory Dashboard

See exactly how much RAM Chrome is using. Visual ring shows usage at a glance.

Heavy Site Detection

Identifies memory-hungry sites like YouTube, Claude, and ChatGPT. Prioritize what matters.

1. Download ZIP

Download the latest source archive from GitHub and unzip it on your machine.

2. Open Extensions

Go to chrome://extensions and switch on Developer mode.

3. Load Unpacked

Select the unzipped cleen-main folder to install Cleen in Chrome.

Stack: Manifest V3, Chrome Processes API, local storage.

Privacy: All data stays on your device. No analytics, no tracking.
