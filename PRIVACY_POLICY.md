# Cleen Privacy Policy

Last updated: March 12, 2026

`Cleen` is a Chrome extension that helps users monitor tab memory usage and suspend inactive tabs.

## Data handling summary

- `Cleen` does not sell user data.
- `Cleen` does not transfer user data to third parties.
- `Cleen` does not use remote analytics, ad networks, or tracking SDKs.
- `Cleen` processes tab and site data locally inside the browser in order to power its core features.

## What data the extension uses

Depending on the page and browser capabilities, `Cleen` may access:

- tab URLs and titles
- tab activity state such as active / discarded status
- locally available memory usage estimates for tabs or processes
- user-defined exclusion rules and extension settings

## Why this data is used

This data is used only to provide the extension's core functionality:

- detect inactive tabs
- estimate memory usage
- identify heavy sites
- suspend or restore tabs on user request
- save local preferences such as exclusions and thresholds

## Where data is stored

All settings and session data are stored locally in Chrome extension storage on the user's device.

## Remote code and external services

`Cleen` does not download or execute remote code.
`Cleen` does not send extension data to external servers.
